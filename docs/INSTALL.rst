sinarngo.pressroom Installation
-------------------------------

To install sinarngo.pressroom using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``sinarngo.pressroom`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        sinarngo.pressroom

* Re-run buildout, e.g. with:

    $ ./bin/buildout
